Frequently asked

How do I add my repo to the allowlist?

Open a PR adding an entry to config/targets.json with the repo URL, ref, category, and a note. Your handle is recorded in the audit log on first submission.

Will the pipeline submit reports for me?

No. The Report stage writes a Markdown file to disk. It never makes HTTP calls to bug-bounty platforms — disclosure is your decision.

What happens if Docker isn't available?

Exploit validation is skipped. The pipeline still proceeds to Patch and Report — the PoC is recorded as unexecuted, and that fact appears in the report.

Which models does it use?

Three tiers (fast / reasoning / coder). The default primary is gemini-3-flash-preview with a fallback chain to gemini-2.5-flash and gemini-2.5-flash-lite. Groq and OpenRouter are optional fallback providers.
